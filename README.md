# Squirrel Stash

An ASP.NET MVC application built to explore object-oriented programming, web application development, and relational databases.

## Overview

The project demonstrates an ASP.NET MVC application for managing squirrels and the items stored in their stashes. The squirrel theme was chosen as part of a school assignment and serves as the domain for practicing CRUD operations, object-oriented design, and relational database integration.

## Features

- Create, edit, and delete squirrels
- Manage items stored within each squirrel's stash
- CRUD operations backed by SQL server
- ASP.NET MVC web interface

## Technologies

- C#
- .NET
- ASP.NET MVC
- SQL Server

## Project Status

This project is archived and no longer maintained. It remains available as a reference for my early work with C#, ASP.NET MVC, and relational database development.
